# Myntra Ads Dashboard

This project is intended to serve as an ads dashboard for Myntra. It will include various analytics and features to help boost advertising efforts.

## Project Structure
- **src/**: Source code of the application.
- **assets/**: Images, logos, and other static assets.
- **documentation/**: Documentation related to the project.
- **tests/**: Automated tests for the application.

## Setup
1. Clone the repository.
2. Install dependencies: `npm install`
3. Run the application: `npm start`
